# ClinicalFlow — Phase 5: PDF Export & File Export

## Overview

Make PDF and text file export work reliably inside Tauri's webview. The existing `exportPDF()` uses `window.open()` to create a popup with `window.print()` — this may not work in Tauri. The fix is to print from the main window using a hidden print container with `@media print` CSS.

**Prerequisites:** Phases 1-4 complete. Note generation works (Cloud AI, Ollama, or Rule-Based). The "Export PDF" and "Copy Note" buttons appear after generating a note.

---

## 1. The Problem

Current code (app.js line 1100):
```javascript
const printWin = window.open('', '_blank', 'width=850,height=1100');
printWin.document.write(html);
printWin.document.close();
```

In Tauri's WKWebView (macOS) or WebView2 (Windows), `window.open()` may be blocked or may open a blank window that can't trigger `window.print()`. We need a Tauri-safe approach.

## 3. Upgrade Text File Export with Save Dialog

Replace the existing `downloadTextNote()` function (app.js, around line 1134) to use Tauri's native save dialog instead of the blob URL hack:

```javascript
async function downloadTextNote() {
  if (!App.noteSections?.sections) return;

  const text = App.noteSections.sections.map(s => {
    const live = document.getElementById(`section-${s.key}`);
    return `=== ${s.title.toUpperCase()} ===\n${live ? live.innerText : s.content}`;
  }).join('\n\n');

  const hdr = `${App.noteSections.title}\nDate: ${fmtDate(App.sessionStartTime || new Date())}\nGenerated by ClinicalFlow\n${'─'.repeat(40)}\n\n`;
  const fullText = hdr + text;
  const defaultName = `ClinicalFlow_Note_${new Date().toISOString().split('T')[0]}.txt`;

  if (window.__TAURI__) {
    try {
      const { save } = window.__TAURI__.dialog;
      const filePath = await save({
        defaultPath: defaultName,
        filters: [{ name: 'Text Files', extensions: ['txt'] }]
      });
      if (filePath) {
        await tauriInvoke('save_text_file', { path: filePath, content: fullText });
        toast('Note saved!', 'success');
      }
      // If filePath is null, user cancelled — do nothing
    } catch (e) {
      toast(`Save failed: ${e}`, 'error');
    }
  } else {
    // Browser fallback (original)
    const blob = new Blob([fullText], { type: 'text/plain' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = defaultName;
    document.body.appendChild(a);
    a.click();
    a.remove();
    URL.revokeObjectURL(url);
    toast('Text file downloaded!', 'success');
  }
}
```

### 3.1 Rust Command for Text Save

Add to `src-tauri/src/storage.rs`:

```rust
#[tauri::command]
async fn save_text_file(path: String, content: String) -> Result<(), String> {
    std::fs::write(&path, &content)
        .map_err(|e| format!("Failed to save file: {}", e))
}
```

Register in `lib.rs`:
```rust
.invoke_handler(tauri::generate_handler![
    // ... existing commands ...
    save_text_file,
])
```

### 3.2 Tauri Plugin Requirement

The save dialog requires `tauri-plugin-dialog`. This should already be installed from Phase 1. Verify in `Cargo.toml`:

```toml

```

And in `capabilities/default.json`:
```json

  "dialog:default",
  "dialog:allow-save"
]
```

## 4. Fallback: Rust PDF Generation (Only If Print Dialog Fails)

If `window.print()` doesn't work in Tauri's webview on your machine, implement programmatic PDF generation as a backup. **Try the print dialog approach first — only do this if it fails.**

Add to `Cargo.toml`:
```toml
genpdf = "0.3"
```

Add to `src-tauri/src/storage.rs`:

```rust
use genpdf::{Document, Margins};
use genpdf::elements::{Paragraph, Break};
use genpdf::style::{Style, StyledString};

#[derive(serde::Deserialize)]
pub struct PdfSection {
    title: String,
    content: String,
}

#[tauri::command]
async fn generate_pdf(
    app: AppHandle,
    title: String,
    date: String,
    duration: String,
    speakers: String,
    sections: Vec<PdfSection>,
) -> Result<String, String> {
    let font_data = include_bytes!("../resources/fonts/LiberationSans-Regular.ttf");
    let bold_data = include_bytes!("../resources/fonts/LiberationSans-Bold.ttf");

    let font_family = genpdf::fonts::FontFamily {
        regular: genpdf::fonts::FontData::new(font_data.to_vec(), None)
            .map_err(|e| format!("Font error: {}", e))?,
        bold: genpdf::fonts::FontData::new(bold_data.to_vec(), None)
            .map_err(|e| format!("Font error: {}", e))?,
        italic: genpdf::fonts::FontData::new(font_data.to_vec(), None)
            .map_err(|e| format!("Font error: {}", e))?,
        bold_italic: genpdf::fonts::FontData::new(bold_data.to_vec(), None)
            .map_err(|e| format!("Font error: {}", e))?,
    };

    let mut doc = Document::new(font_family);
    doc.set_title(&title);
    doc.set_margins(Margins::trbl(36, 54, 36, 54));

    // Header
    doc.push(Paragraph::new(StyledString::new(
        "ClinicalFlow — Clinical Documentation",
        Style::new().with_font_size(9),
    )));
    doc.push(Paragraph::new(StyledString::new(&title, Style::new().bold().with_font_size(16))));
    doc.push(Paragraph::new(format!("Date: {} | Duration: {} | Speakers: {}", date, duration, speakers)));
    doc.push(Break::new(1.0));

    // Disclaimer
    doc.push(Paragraph::new(StyledString::new(
        "This note was auto-generated from a transcribed clinical encounter. It should be reviewed, verified, and amended by the treating provider before inclusion in the medical record.",
        Style::new().with_font_size(8),
    )));
    doc.push(Break::new(1.0));

    // Sections
    for section in &sections {
        doc.push(Paragraph::new(StyledString::new(
            &section.title.to_uppercase(),
            Style::new().bold().with_font_size(11),
        )));
        // Split content into paragraphs
        for line in section.content.lines() {
            if line.trim().is_empty() {
                doc.push(Break::new(0.3));
            } else {
                doc.push(Paragraph::new(line));
            }
        }
        doc.push(Break::new(0.5));
    }

    // Footer
    doc.push(Break::new(1.0));
    doc.push(Paragraph::new(StyledString::new(
        "Generated by ClinicalFlow — For review by treating provider",
        Style::new().with_font_size(8),
    )));

    // Save
    let base = app.path().app_data_dir().map_err(|e| e.to_string())?;
    let exports_dir = base.join("exports");
    std::fs::create_dir_all(&exports_dir).map_err(|e| e.to_string())?;

    let filename = format!("ClinicalFlow_Note_{}.pdf",
        chrono::Local::now().format("%Y-%m-%d_%H-%M"));
    let path = exports_dir.join(&filename);

    doc.render_to_file(&path)
        .map_err(|e| format!("PDF render failed: {}", e))?;

    Ok(path.to_string_lossy().to_string())
}
```

```javascript
// Only use this if window.print() doesn't work in Tauri
async function exportPDFViaRust() {
  if (!App.noteGenerated) { toast('Generate a note first.', 'warning'); return; }

  const sections = App.noteSections.sections.map(s => {
    const live = document.getElementById(`section-${s.key}`);

      sections: sections

  } catch (e) {
    toast(`PDF export failed: ${e}`, 'error');
  }
}
```
